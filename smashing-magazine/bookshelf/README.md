# Bookshelf

This is a demo for the article --- published on [Smashing Magazine](https://smashingmagazine.com).

## Getting Started

1. Clone the repo
2. Run the command `yarn install` to install all packages
3. Run the command `yarn dev` to run the project

## Tech Stack

1. Next.js
2. SASS/SCSS
3. CSS Modules
4. Emotion
5. Styled Components
6. Styled JSX
